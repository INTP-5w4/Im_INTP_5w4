-d sirve para eliminar muchas cosas, como tag, brannch y demás
cd Moverse a la carpeta
../ Volver una carpeta
git status indica estado
mkdir  sirve para crear carpetas
Git commit -a sube todo de una
Git mv sirve para renombrar un archivo
git diff muestra la diferencia
checkout regresa a versiones anteriores
Git log Muestra los cambios
Para ignorar ciertos archivos crear un documento de texto con extension .gitignore
Git branch (nombre rama) Para crear nueva rama
git log --oneline --decorate --graph Sirve para mostrar todo de forma resumida
Para cambiar de rama se usa Git checkout (nombre de la rama)
Git merge (nombre de rama a unir(debes estar en la rama principal)) para unir ramas
Git branch -d Borrar rama
Git checkout -b (Nombre rama) Para crear y dirigirnos a una rama en especifico
Git tag (Nombre etiqueta) Sirve para etiquetar los commits
Git tag -a (Nombre etiqueta) codigo commit Sirve para etiquetar los commits a voluntad
git push origin main sirve para subir archivos locales al repositorio
git remote add origin https://github.com/INTP-5w4/Im_INTP_5w4.git sirve para vincular un repositorio.
Git pull Sirve para descargar archivos en linea
